3. Réalisation

Environnement Technique

Core Logic

Architecture basée sur des Modules ES6 pour séparer la vue (DOM) de la logique métier (Services).

UI / UX

Utilisation de variables CSS (:root) pour une maintenance facile du thème (Couleurs, Spacing).

Libs Externes

Chart.js pour la dataviz et jsPDF pour la génération administrative.

i18n

Système de dictionnaire JSON pour le changement de langue dynamique (FR/EN/AR).
